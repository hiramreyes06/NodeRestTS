Primero necesitamos subir el proyecto de node compilado a .js, con el package 
json, package-lock.json, con el comando, para que nos compile a js dist/

tsc -w

Ahora en la carpeta dist/ nos aseguramos de que tenga el package.json y el
package-lock.json , donde en el package.json debemos agregarle scripts
main, serve, start, engines con node DEPENDIENDO de donde inicie mi
servidor en el codigo

{
  
  "name": "socket-server",
  "version": "1.0.0",
  "description": "Rest server para mapbox",
  "main": "index.js",
  "scripts": {
    "serve":"node index.js",
    "start": "node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "engines" : { 
    "node" : ">=8.0.0 <11.0.0" 
  },
